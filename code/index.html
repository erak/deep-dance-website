Overview

The set of software components of Deep.Dance contains a system, that uses to machine learning to study exisiting and create new movement sequences. In order to visualize these newly created movement sequences a web-based component, that could render them, was added. An additional web-based component, that can control and animate a LED light setup on stage, was also added.
